Development environments

This website shows the contents of the Development Environment module, which is part of the Multi-platform Application Development and Web Application Development professional training courses. This module is about software engineering and some useful tools and techniques, such as version control tools, software testing and so on.

Besides, we also have a complete overview of Java programming language, which is the language we're going to rely on along the course. You can have a look at the complete Java course here.
